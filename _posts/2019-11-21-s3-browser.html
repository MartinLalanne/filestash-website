Browse, Explore and Manage your S3 buckets

Our web client makes it easy to upload files to S3 via drag and drop but also view those files individually, rename them, edit a range of file formats and delete anything all from the web interface

Amazon S3 but not only

Wether you're using AWS or an API compatible alternative, Filestash is there to help you explore your buckets. As a example, you can connect to Minio

Mac, Windows, Linux, IOS and Android

Filestash is the missing S3 app that's is available from your browser. No download required, our tool is portable and works on any platform

Open source and Free software

You don't have to trust us. The code of our software is available on github for everyone to download, audit, selfhost and contribute back

Document editor

Hosting a static website out of your S3 bucket? Or just storing a lot documents? You can edit anything you want from word documents spreadsheets, presentations, html files, css files and everything in between

Collaborate easily

Filestash lets you create shared links without sharing your keys. Shared links can be protected with a password and/or only available to a specific email adress or a domain (example: '*@my-company.com')

Easy to use

Filestash intend to be easy to use by everyone without training. Power users who needs extra power can always use the complete AWS CLI to access all advanced features built for them.

File manager integration

Shared links can be mounted in your file manager and be used like any other network drive

Full text search

Find your content is key. Filestash has a powerfull full text search engine. It will create its own index automatically by crawling your buckets (The feature is disable by default to avoid extra charge from Amazon to crawl your bucket)

Extensible and Configurable

Filestash can be configure to predefine login information and wide range of other features that can be enable/disable. Also you can extend the program by creating your own plugins

Creating an awesome S3 Browser

Traditional S3 GUI and CLI such as the Amazon CLI, Cloudberry Explorer or Cyberduck are great for sys admin who need the full power of S3, but having access to all those features makes it harder to use for everyone else. The value proposition of Filestash is to make protocols and platform like Amazon S3 easy to use for the end users (not sys admins who already have great tools to work with) bringing collaboration features and making S3 look and feel like a modern Dropbox replacement
